# Full Stack E-Commerce Platform

## Overview
A full-stack e-commerce web application built with modern web technologies.  
It supports authentication, product management, cart system, and order flow with a scalable architecture.

---

## Features

### User Features
- Browse products by categories
- Add/remove products to cart
- Place orders
- View order history

### 🛠 Admin Features
- Add / update / delete products
- Manage categories
- Manage orders
- Manage Sales

---

## Tech Stack
- Frontend: Next.js
- Backend: Node.js / Express
- Database: MySQL
- Authentication: JWT
- API: RESTful APIs

---

## Architecture
- Frontend communicates via REST APIs
- Backend handles business logic
- Database stores users, products, orders, cart
- JWT used for secure authentication
